组合技远程唤醒加远程控制（使用SS Server作为VPN网关）

有的时候，回到了家，突然发现今天更新的代码临走的时候没有推送到git，跑回公司去开机push这种低端的事情绝对做不出。那么怎么处理？

这个工作我们需要组合几个功能：

  1. SS Server ，用作VPN Server接入公司内网
  2. Teamview，远程桌面神器
  3. 老毛子固件的WOL网络唤醒功能
  4. 老毛子固件的DDNS功能

我们看看具体的操作步骤吧：

  1. 首先，固件上需要启动SS -Server
  2. 然后，注册一个DDNS服务器，我用的是dynu.com 的服务器，一个账号可以注册4台设备，也就是4个宽带，刚好覆盖家、父母家、公司、岳父家....
  3. 电脑打开远程唤醒功能：

    windows 电脑要看主板，部分bios带有支持WOL开机的，打开那个选项便可。休眠状态需要升级网卡驱动，在驱动的高级设置里面打开具体教程自行google “win10 wol ”之类的关键字吧。
  4. 电脑装上ss客户端，加入公司的ss服务器配置
  5. 公司电脑安装好Teamviewer，设置好无人值守
  6. 开启电脑上的ss客户端，接入公司ss服务器，打开teamveiwer，准备看到公司的电脑上线

  7. 电脑打开浏览器， 输入要 my.router，便可打开公司路由器的管理界面，然后唤醒你的电脑

    点击唤醒以后，大约10~20秒（视你机器开机或者唤醒速度而定），可以看到公司的电脑在teamviewer上线了。

  8. 点击链接 teamviewer，收工。

 

这个教程其实讲的并不是远程开机，而是灵活使用ss server 作为企业内网的VPN网关，只要你家里的网段和公司的网段不同，可以非常简单的访问公司内网，可以直接升级路由固件什么的。当然，前提是你的宽带有公网IP。
